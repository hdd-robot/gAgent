Detailed Description

Thread-safe logger singleton.

Provides centralized logging functionality with multiple severity levels, file output support, and timestamp formatting.

Example usage:

Logger::getInstance().log(LogLevel::INFO, "Agent started", "MyAgent");
Logger::getInstance().setLogFile("agent.log");
