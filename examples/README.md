# Examples

This directory contains example programs demonstrating gAgent usage.

## improved_agent

Modern C++ example with logging, error handling, and best practices.

### Build

```bash
cd build
cmake ..
make improved_agent
```

### Run

```bash
# Console mode
./examples/improved_agent

# With GUI
./examples/improved_agent --gui
```

### Features demonstrated

- Logger system usage
- Error handling with custom exceptions
- Modern C++ practices
- Custom behaviours
- Environment integration
- Agent lifecycle management

## Output

The example creates a log file `improved_agent.log` with detailed execution information.
